# ZoomLens

🇷🇺 Русский | 🇬🇧 [English](README.md)

**ZoomLens** — экранная лупа для Windows.

[Скачать ZIP](https://github.com/ZoomLensApp/ZoomLens/releases/download/v1.1-beta/ZoomLens-1.1-beta-portable.zip) · [Все релизы](https://github.com/ZoomLensApp/ZoomLens/releases)

Программа показывает увеличенную область экрана в виде круглой линзы поверх рабочего стола или в отдельном окне. Это может помочь читать мелкий интерфейс, рассматривать детали на экране или работать с отдельной областью без изменения масштаба всей Windows.

> ZoomLens 1.1 Beta — ранняя публичная бета-версия. Некоторые ограничения ожидаемы.

## Возможности

* Круглая линза поверх экрана
* Отдельное окно увеличения с навигатором
* Выбор монитора для захвата и вывода
* Режимы следования за курсором
* Глобальные горячие клавиши
* Компактная панель управления
* Интерфейс на английском и русском языках
* Portable-конфигурация через локальный `config.ini`

## Как пользоваться

1. Скачайте portable ZIP со страницы релизов.
2. Распакуйте архив в любую папку.
3. Запустите `ZoomLens.exe`.
4. Настройте режим линзы в панели управления.
5. Нажмите **Save Changes**, чтобы сохранить настройки.

## Конфиденциальность

ZoomLens работает локально на вашем компьютере.

Программа не собирает телеметрию, не записывает экран, не делает скриншоты и не отправляет содержимое экрана куда-либо.

Подробнее: [PRIVACY.md](PRIVACY.md)

## Известные ограничения

* Это beta-версия.
* Приложение пока не подписано цифровой подписью, поэтому Windows SmartScreen может показать предупреждение.
* Некоторые игры в полноэкранном режиме могут скрывать или блокировать оверлеи.
* Для лучшей совместимости с играми обычно подходит режим окна без рамки / borderless windowed.
* Режим отдельного окна лучше всего использовать с двумя мониторами.
* Если изменить масштаб Windows во время работы ZoomLens, перезапустите программу.
* При изменении конфигурации мониторов во время работы ZoomLens может потребоваться перезапуск приложения.
* Сглаживание изображения запланировано для будущей версии движка.
* Чёткость изображения зависит от исходного разрешения и масштаба.

## Игры и античит

ZoomLens не внедряется в игры. Программа использует стандартные механизмы окон Windows и Windows Magnification API.

Однако некоторые игры, полноэкранные режимы или античит-системы могут блокировать оверлеи или работать с ними нестабильно. Используйте ZoomLens в соревновательных играх на своё усмотрение.
