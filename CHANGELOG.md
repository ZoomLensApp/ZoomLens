# Changelog

## ZoomLens 1.1 Beta

Initial public beta release.

### Added

- Overlay Lens mode
- Separate Window mode
- Navigator for Separate Window mode
- Source and output monitor selection
- Follow Cursor modes
- Global hotkeys
- English and Russian interface
- Help section
- Feedback section
- Support section
- Portable configuration via `config.ini`

### Notes

- This is a beta release.
- Some fullscreen games may hide or block overlays.
- Borderless windowed mode is recommended for better compatibility.
- Separate Window mode works best with two monitors.
- Smooth interpolation is planned for a future engine version.
- The beta build may be unsigned, so Windows SmartScreen may show a warning.
